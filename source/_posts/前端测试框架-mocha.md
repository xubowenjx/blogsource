## mocha 前端测试框架

### 安装mocha
首先需要一个nodejs环境,比如说新建一个工程mocha-test
```bash
mkdir mocha-test
cd mocha-test
npm init
npm install -s mocha
```
为了方便可以全局安装
```bash
npm install -g mocha
```
